# V2X Collision Warning ASIC 🚗⚠️

A TinyTapeout/OpenLane based ASIC design for Vehicle-to-Everything (V2X) collision warning detection using Verilog HDL.

## 📌 Project Overview
This project detects potential collision conditions between vehicles using digital logic implemented in Verilog.  
The design is synthesized and converted into a physical ASIC layout using the OpenLane RTL-to-GDSII flow.

## 🛠 Tools Used
- Verilog HDL
- OpenLane
- Sky130 PDK
- TinyTapeout flow
- GitHub Actions

## 🚀 RTL to GDS Flow
1. Verilog RTL Design
2. Synthesis
3. Floorplanning
4. Placement
5. Clock Tree Synthesis
6. Routing
7. DRC/LVS Checks
8. GDSII Generation
